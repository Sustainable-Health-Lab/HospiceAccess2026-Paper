# HospiceAccess2026-Paper
Code and Data for "Close but Unreachable: Racial Disparities in Hospice Utilization Beyond Geographic Proximity"



## 1-Data

The following datasets were used for statistical analyses:

- **hospice_care_2018/**  
  Folder containing archived data snapshots from the Centers of Medicare and Medicaid Services on Medicare-
certified hospice agencies in 2018.
Downloaded from: https://data.cms.gov/provider-data/archived-data/hospice-care

- **2018CensusIncome.zip**  
  Compressed file with 2018 U.S. Census income data.
  Downloaded from: https://data.census.gov/table/ACSST5Y2018.S1902?q=Income+by+Zip+code+tabulation+area.

- **All_geocoded_addresses.csv**  
  CSV file containing geocoded hospice agency addresses which were computed using ArgGIS.
  

- **RUCA2010zipcode.csv.zip**  
  Compressed file with 2010 Rural-Urban Commuting Area (RUCA) codes by ZIP code from U.S. Department of Agriculture's Economic Research Service.
  Downloaded from: https://www.ers.usda.gov/data-products/rural-urban-commuting-area-codes
